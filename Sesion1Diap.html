🔍 ¿QUÉ ES ANALÍTICA DE DATOS?

La analítica de datos es "el proceso de examinar conjuntos de datos para descubrir patrones, extraer insights y apoyar la toma de decisiones" (Chen et al., 2012).

🔬📊🧠

Componentes Clave:

  • Datos: Información en bruto
  • Procesamiento: Limpieza y transformación
  • Análisis: Identificación de patrones
  • Insights: Conocimientos accionables
(Chen, H., Chiang, R. H., & Storey, V. C., 2012)

📁 TIPOS DE DATOS

🗃️ ESTRUCTURADOS

Organizados en formato tabular

Ejemplos: Bases de datos, Excel, CSV

📊

📄 NO ESTRUCTURADOS

Sin formato predefinido

Ejemplos: Texto, imágenes, videos

📸

Según IBM (2023), el 80% de los datos del mundo son no estructurados, lo que representa un gran desafío y oportunidad.

(IBM, 2023)

⚡ POWER BI

📊💼

¿Qué es Power BI?

Plataforma de Business Intelligence de Microsoft que permite conectar, transformar y visualizar datos de múltiples fuentes (Microsoft, 2024).

🚀 Componentes Principales

  • Power Query: Transformación de datos
  • DAX: Lenguaje de fórmulas
  • Power BI Service: Plataforma en la nube
  • Visualizaciones: Gráficos interactivos

📈 R PARA ESTADÍSTICA

📊🔬

R es "el lenguaje estadístico por excelencia", especialmente potente para análisis estadístico avanzado y visualizaciones (Wickham & Grolemund, 2023).

🌟 Fortalezas de R

  • ggplot2: Visualizaciones elegantes
  • dplyr: Manipulación de datos
  • RStudio: IDE profesional
  • CRAN: 18,000+ paquetes especializados

🔄 METODOLOGÍA CRISP-DM

Cross-Industry Standard Process for Data Mining - metodología estándar para proyectos de analítica (Chapman et al., 2000).

1️⃣ Comprensión del Negocio

2️⃣ Comprensión de Datos

3️⃣ Preparación de Datos

4️⃣ Modelado

5️⃣ Evaluación

6️⃣ Despliegue

(Chapman, P., et al., 2000)
